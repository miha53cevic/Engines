TEXTURES
- Smart Unwrap UV and Enable Live Unwrap
- UV tab then UV export to get a picture with the model UVs

extra
- if you want to see the texture go to shading
- click on add/texture/image then connect color with the UV colour
- Hold Z and change rendering mode to see the texture
